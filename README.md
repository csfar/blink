# Blink

Carry out your brainstormings with Blink! - Brainstorming. With it you will be able to connect with up to 7 iPhones and conduct a complete brainstorming session.

## Features

* Timer
* Multipeer
* Brainstorming
  * Add ideas (tvOS)
  * Add ideas (iOS)
* Voting
  * Vote on ideas (tvOS)
  * Vote on ideas (iOS)
* Ranking
  
## Users

### Mediator
* tvOS
* Siri Dictation
* Controls the session
* Can add ideas
* Can vote on ideas

### Participants
* iOS
* Multipeer
* Can add ideas
* Can vote on ideas
