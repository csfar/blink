# Blink

## Features

* Timer
* Multipeer
* Brainstorming
  * Add ideas (tvOS)
  * Add ideas (iOS)
* Voting
  * Vote on ideas (tvOS)
  * Vote on ideas (iOS)
* Ranking
  
## Users

### Mediator
* tvOS
* Siri Dictation
* Controls the session
* Can add ideas
* Can vote on ideas

### Participants
* iOS
* Multipeer
* Can add ideas
* Can vote on ideas
